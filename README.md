# My Portfolio Website

Welcome to the repository for personal portfolio website! This project showcases  skills in HTML, CSS, and JavaScript and serves as a platform to display projects and information.

## Technologies Used

- HTML5
- CSS3
- JavaScript

- To run this portfolio website locally, follow these steps:

1. Clone the repository: `git clone https://github.com/yourusername/your-portfolio.git`
2. Open the project folder in your preferred code editor.
3. Open the `index.html` file in your web browser.

4. Happy coding
